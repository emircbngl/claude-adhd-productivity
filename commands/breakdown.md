description: Break a task into 2-minute next actions.

Read the `adhd-focus` skill and `references/task-decomposition.md`, then break down the task.

Task: $ARGUMENTS
(If empty, use the Current task in `FOCUS.md`.)

1. Name the finish line in one sentence.
2. List 3–6 rough chunks (don't over-engineer).
3. Find the first 2-minute physical action and put it in the NOW line of `FOCUS.md`.
4. Write the step list to `FOCUS.md` with "← you are here" on the current step.
5. In your reply, show ONLY the next single action — the full list lives in the file so it doesn't recreate overwhelm.

# INCUP — manufacturing motivation for an interest-based brain

The ADHD brain is driven by **INCUP**, not by importance: **I**nterest, **N**ovelty, **C**hallenge, **U**rgency, **P**assion (framework from Dr. William Dodson). When a task has none of these, starting it feels viscerally hard — that's neurology, not a willpower failure. Your move: when a task is flat, **add one or two INCUP levers**. You rarely need all five.

## The five levers and how to add them

### Interest
Tie the task to something the user actually cares about, or let them start with the part that's interesting.
- "Which part of this is least boring? Start there."
- Pair a dull task with something engaging (music, a favorite drink, a nice doc template).

### Novelty
The brain perks up at *new*. Change something.
- New environment: different room, café, standing up.
- New tool/format: dictate instead of type, whiteboard it, use a fresh doc, a different color.
- New framing: "do it the weird way" — write the report as a text to a friend first.

### Challenge
Turn it into a game against yourself.
- "Bet you can't get the first paragraph done before the timer."
- Add a constraint: "in exactly 5 bullet points," "using no adjectives," "one-handed."
- Track a high score (steps done, words, minutes focused).

### Urgency  ← usually the strongest, fastest lever
The ADHD brain hyperfocuses under time pressure. Manufacture it artificially when it's missing.
- **Set a timer** and say "go" — a countdown creates real urgency where importance can't.
- Mini-deadline: "Let's get just this one step done in the next 10 minutes."
- "Race the clock" framing; sprint, then stop.

### Passion
Connect to the why — values, future self, the feeling of being done.
- "Once this is sent, it's off your plate for good — imagine that."
- "Future-you at 6pm will be so relieved this is handled."
- Link to identity/values: "this is the kind of person you want to be."

## How to use it in practice

1. Notice flatness — the user stalls, sighs, "ugh I don't want to."
2. Pick the **1–2 easiest levers** to add right now. Urgency (a timer) is almost always available and almost always works.
3. Reframe out loud, then immediately point at the NOW action.

**Example:** "Reconcile the spreadsheet" (zero INCUP) →
"Okay, this one's boring, so let's make it a race: 12-minute timer, see how many rows you clear before it goes off. Winner gets to stop guilt-free. Open the sheet — go."

(Urgency + Challenge added; activation energy removed; one action given.)

## Don't overdo it

The point is to get moving, not to gamify everything forever. If the user is already engaged or in flow, **don't interrupt to add levers** — just stay out of the way and let hyperfocus run.

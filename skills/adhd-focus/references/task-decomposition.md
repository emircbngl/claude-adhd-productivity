# Task decomposition — shrink to the 2-minute next action

The single highest-leverage thing you do. A big task is a wall; a 2-minute step is a door. The ADHD brain can't start "write the report" but it can "open the doc and type a title."

## The core question

For any task, ask: **"What is the very first *physical* action — something I could do in about 2 minutes without thinking?"**

Then put *that* in the NOW line. Everything else waits.

## Rules

- **Start with a physical verb.** Good: open, type, write one sentence, click, copy, stand up, find, paste, rename, run. Bad (too abstract — these are not actions): plan, think about, work on, organize, finish, handle, deal with, look into, start.
- **2 minutes, not 20.** If the first step would take more than ~2–5 minutes, it's still too big. Split again.
- **One step visible at a time.** Write the full step list to `FOCUS.md`, but in conversation only point at the current one. Showing all ten steps re-creates the overwhelm you just removed.
- **Concrete beats complete.** A slightly wrong tiny step that gets them moving beats a perfect plan that doesn't. Momentum first.
- **Stop decomposing when the first step is genuinely doable right now.** You don't need to plan the whole task — just the next door.

## How to split

1. Name the finish line in one sentence ("the leak email is sent").
2. List the 3–6 chunks roughly (don't over-engineer).
3. Take the **first** chunk and find its first 2-minute physical action.
4. Put that one action in NOW. Done — start there.

## Examples

**"I need to do my taxes"** →
- NOW: Open the folder where last year's tax PDF is.
- then: Find this year's income doc. Open the tax site. Log in. (one at a time)

**"Write the project proposal"** →
- NOW: Open a new doc and type the project name as the title.
- then: Write one ugly sentence describing the goal. (We'll fix it later — see anti-shame-rsd.md.)

**"Clean the apartment" (huge, vague)** →
- NOW: Set a 10-min timer and pick up only things off the floor in the living room.
- Parking lot: kitchen, bathroom, laundry — not now.

**"Reply to all these emails"** →
- NOW: Open the *oldest* one and read it. (Don't decide about all of them — just open one.)

## When they're stuck even on the small step

It's still too big, or there's avoidance. Shrink again — make it almost embarrassingly small ("just open the file; you don't have to write anything") — and consider body-doubling it (see `body-doubling.md`). If there's emotional charge, route through `anti-shame-rsd.md` first.

## The 2-minute rule (the other half)

If something genuinely takes **less than 2 minutes**, don't schedule it — do it now. Use this to clear quick wins and build momentum at the start of a session.
